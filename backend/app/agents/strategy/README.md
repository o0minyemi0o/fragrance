# Strategy Agents - 전략 수립

## 📋 역할
생성된 배합과 시장 조사 결과를 바탕으로 **포지셔닝**과 **가격 전략**을 수립합니다.

## 📁 파일 구조
- **positioning_agent.py**: 제품 포지셔닝
- **pricing_agent.py**: 가격 전략

## 🎯 주요 기능

### positioning_agent.py
**역할**: 제품 차별화 및 브랜드 메시지 제안

**입력**:
- 배합 특성 (프레시 플로럴, 지속력 6-8시간)
- 시장 트렌드 (Clean Beauty 트렌드)
- 타겟 고객 (30대 여성, 활동적)

**출력**:
- **포지셔닝 메시지**: "바쁜 일상 속 상쾌한 에너지"
- **차별화 요소**: 천연 원료 80% 사용
- **마케팅 앵글**: "아침부터 저녁까지 지속되는 상쾌함"

---

### pricing_agent.py
**역할**: 최적 가격 전략 수립

**입력**:
- 원가 (12,000원)
- 경쟁사 가격 (25,000 ~ 40,000원)
- 타겟 가격대 (30,000원)

**출력**:
- **권장 가격**: 32,000원
- **마진**: 약 62%
- **가격 정당화**: "천연 원료 사용, 8시간 지속력"
